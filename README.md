# TechShop
🖥️ Full-stack e-commerce platform for tech products. Modern UI, REST API, secure checkout &amp; real-time inventory.
📌 Présentation
TechShop est une plateforme e-commerce complète permettant aux utilisateurs de parcourir, rechercher et acheter des produits technologiques. Le projet inclut une interface utilisateur moderne, un système de panier, une gestion des commandes et un back-office d'administration.

✨ Fonctionnalités

🔍 Recherche & filtres avancés — par catégorie, prix, marque
🛍️ Panier & commande — ajout, suppression, validation de commande
👤 Authentification — inscription, connexion, gestion de profil
📦 Gestion des produits — catalogue dynamique avec fiches détaillées
📱 Design responsive — adapté mobile, tablette et desktop
🔒 Paiement sécurisé — intégration passerelle de paiement
🛠️ Back-office admin — gestion des produits, commandes et utilisateurs
